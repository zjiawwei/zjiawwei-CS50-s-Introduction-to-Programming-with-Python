## 📖 关于本项目

这个仓库是我在学习哈佛大学 **CS50’s Introduction to Programming with Python** 课程期间完成的作业和练习项目合集。CS50 的 Python 课程是哈佛大学著名的 CS50 系列课程之一，专注于教授 Python 编程语言的基础知识和应用。

### 🎯 课程目标
- 掌握 Python 编程语言的基础语法
- 理解编程核心概念：变量、函数、条件语句、循环等
- 学习使用第三方库和 API
- 培养解决问题的能力
- 建立良好的代码编写习惯

## 🛠️ 开发环境配置

### 本地开发环境（WSL Debian + VS Code）

本项目在 **WSL (Windows Subsystem for Linux)** 环境中开发，使用 **Debian** 发行版和 **Visual Studio Code** 编辑器。
